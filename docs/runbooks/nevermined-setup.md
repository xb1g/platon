# Nevermined Setup

1. Create an account on Nevermined
2. Get your RPC URL and Wallet Private Key
3. Add them to your `.env` file:
   ```
   NEVERMINED_RPC_URL=your_rpc_url
   NEVERMINED_WALLET_PRIVATE_KEY=your_private_key
   ```
4. The API server will automatically meter usage for protected endpoints.
